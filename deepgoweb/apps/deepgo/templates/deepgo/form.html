DeepGO Protein Function prediction.


DeepGO is a novel method for predicting protein functions using protein sequences. It uses deep neural networks to learn sequence features and hierarchically classifies it with GO classes.

We also have a method which incorporates protein-protein interaction networks and performs significantly better than sequence model. The source code of the model is available on GitHub repository


Examples: AAK1 (Q2M2I8) PPP2R5D (Q14738)
