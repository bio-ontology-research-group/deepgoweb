Changelog

Platform updates

2026-06 — DeepGO-PlusPlus-Light, predictor API & SPARQL, new look

  • New CPU-only model DeepGO-PlusPlus-Light — a multi-evidence ensemble (DIAMOND, STRING-Net, a hierarchy-aware CNN, ESM2-35M embedding kNN and hierarchy-aware head, ProteInfer) selectable on the Prediction page, with each component's own output shown on the result page.
  • New predictor-aware REST endpoint /deepgo/api/predict (takes a predictor field and returns per-component substreams); the original /deepgo/api/create is unchanged.
  • New SPARQL functions dg:predict and dg:components on the SPARQL endpoint, in addition to the unchanged dg:deepgo.
  • Result pages now show only the most specific GO terms by default (redundant ancestors hidden; full set kept in the JSON/CSV export), and obsolete predicted terms are transferred to their replacements.
  • Modernised the website (Bootstrap 5) and refreshed the security dependencies.
